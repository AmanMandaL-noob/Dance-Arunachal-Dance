# Dance Arunachal Dance — Vite dev

This workspace now contains a minimal Vite + React setup to run `dance-arunachal-dance` locally.

Quick start (run in the project root):

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

If you prefer the original CDN demo (no Node required), open `index.cdn.html` in your browser.
